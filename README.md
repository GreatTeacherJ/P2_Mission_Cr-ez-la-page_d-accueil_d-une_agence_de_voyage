# booki-starter-pack


Site web de recherche d'hébergements et d'activités touristiques.
Projet réalisé dans le cadre de la formation OpenClassrooms.

---

## Description

Booki est une interface web permettant aux utilisateurs de trouver des
hébergements et des activités dans la ville de leur choix.
Ce projet est une intégration HTML/CSS à partir de maquettes Figma.

---

## Technologies utilisées

- **HTML5**
- **CSS3** (Flexbox, Grid, Media Queries)
- **Font Awesome 6** - Icônes
- **Google Fonts** - Police Raleway

## Fonctionnalités

-  Barre de recherche par ville
-  Filtres par catégorie (Économique, Familial, Romantique, Nos pépites)
-  Section hébergements en grille
-  Section "Les plus populaires"
-  Section activités
-  Navigation avec liens d'ancrage
-  Footer avec liens

---

## Responsive Design

Le site est adapté pour 3 tailles d'écran :

| Breakpoint | Taille |
|---|---|
| Desktop | > 1024px |
| Tablette | 768px - 1024px |
| Mobile | < 768px |
